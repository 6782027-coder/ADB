# ADB אלחוטי - הוראות בנייה

פרויקט Android Studio מלא. האפליקציה מפעילה ADB דרך WiFi על מכשיר עם root,
ומציגה את כתובת ה-IP והפורט (5555) להתחברות מהמחשב.

## איך לבנות APK

1. התקן Android Studio (חינמי): https://developer.android.com/studio
2. פתח את Android Studio → Open → בחר את התיקייה הזו (WirelessAdbEnabler)
3. חכה שה-Gradle יסיים לסנכרן (בפעם הראשונה זה מוריד תלויות - צריך אינטרנט)
4. מלמעלה: Build → Build Bundle(s) / APK(s) → Build APK(s)
5. ה-APK ייווצר בנתיב:
   app/build/outputs/apk/debug/app-debug.apk
6. העבר את הקובץ למכשיר (USB / Google Drive / קישור) והתקן אותו
   (יידרש לאשר "התקנה ממקורות לא ידועים")

## איך זה עובד

האפליקציה מריצה את הפקודות הבאות עם הרשאת root במכשיר עצמו:

    setprop service.adb.tcp.port 5555
    stop adbd
    start adbd

זה בדיוק מה ש-`adb tcpip 5555` עושה כשהמכשיר מחובר ב-USB - רק שכאן
זה קורה מתוך המכשיר עצמו, בלי צורך בכבל.

לאחר ההפעלה, במחשב מריצים:

    adb connect <IP-שמוצג-באפליקציה>:5555

## חשוב לדעת

- נדרש מכשיר עם root מלא (Magisk וכו') שמאשר בקשות su מאפליקציות.
- המכשיר והמחשב חייבים להיות על אותה רשת WiFi.
- לאחר reboot של המכשיר, ה-ADB חוזר להיות כבוי (זו התנהגות סטנדרטית) -
  צריך להפעיל שוב מהאפליקציה.
- זהו כלי פיתוח לשימוש אישי במכשיר שלך. אל תשאיר ADB פתוח דרך WiFi
  על רשתות ציבוריות/לא מהימנות - זו גישת שורש למכשיר בלי אימות נוסף.
